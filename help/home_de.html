Plugin "GeODin QGIS"

Das Plugin GeODin QGIS ermöglicht die Nutzung von GeODin-Datenbanken in QGIS.

Die erste Version mit den aus dem Konzept erarbeiteten Basis-Funktionen wurde beendet und kann als Alpha-Version getestet werden.

Das Plugin ist in einem Ordner hinterlegt, der komplett in die QGIS-Konfiguration kopiert bzw. dort abgelegt werden muss.

Nach Absprache mit der QGIS Community werden wir die offizielle Version mit dem Namen „GeODin QGIS“ herausgeben.

Dieser so genannte ‚ GeODinQGIS ‘ Ordner muss unter folgendem Pfad in der QGIS-Konfiguration abgelegt werden.

C:\Benutzer\Benutzername\.qgis2\python\plugins

Zum Beispiel:

C:\Benutzer\MustermannM\.qgis2\python\plugins

Wenn es dabei Probleme, weil die Ordnerstruktur z. B. nicht existiert, kann das Plugin auch im Installationsverzeichnis von QGIS abgelegt werden.

Zum Beispiel:

C:\Program Files (x86)\QGIS Wien\apps\qgis\python\plugins

In QGIS wird das Plugin über das Hinzufügen der Erweiterung eingebunden. Dazu wird in der Menüleiste der Eintrag „Erweiterungen“ gewählt und unter dem Punkt „Erweiterungen verwalten und installieren“ das Plugin in der Liste ausgewählt und aktiviert.
